# Magic Mirror v1.0.2

This release focuses on making the dictionary workflow usable even when the
external gpt-oss-120b gateway is unavailable.

## Fixes

- Replaced the raw dictionary result label with structured right-to-left cards.
- Isolated English and technical tokens such as `HTTP`, `GatewayBaseUrl`, and
  `gpt-oss-120b` so they no longer break Arabic line order.
- Shows gateway failures as a clear status card instead of an unformatted error
  paragraph.
- Keeps dictionary lookup honest: when the gateway returns `502 BadGateway`, the
  app reports that the AI analysis is unavailable rather than fabricating
  alternatives.

## Download

Use `MagicMirror-v1.0.2-windows-x64.zip` from the release assets, extract it,
and run `MagicMirror.Native.exe`.
