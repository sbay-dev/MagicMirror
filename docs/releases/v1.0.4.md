# Magic Mirror v1.0.4

This release connects Magic Mirror to its own dedicated Cloudflare Sarmad
gateway.

## Added

- Published the dedicated Worker gateway:
  `https://magicmirror-sarmad-gateway.2sa.workers.dev`
- The native app now defaults `GatewayBaseUrl` to that Worker.
- Existing blank gateway settings are migrated to the dedicated gateway.
- The Worker uses Cloudflare Workers AI through the server-side `AI` binding and
  defaults to `@cf/openai/gpt-oss-20b`.

## Verified

- `GET /api/sarmad/health` returned `200`.
- `POST /api/sarmad/ask` returned a real Arabic dictionary analysis from the
  deployed Worker.

## Download

Use `MagicMirror-v1.0.4-windows-x64.zip` from the release assets, extract it,
and run `MagicMirror.Native.exe`.
