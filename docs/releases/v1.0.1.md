# Magic Mirror v1.0.1

Maintenance release for the dictionary/selection workflow.

## Fixes

- The dictionary card now has reliable arrow controls and a larger drag handle,
  so it can be moved away from the text being inspected.
- Selecting Arabic rendered text now keeps an exact term/line highlight and
  outlines the corresponding source/document block without covering the page
  with a large yellow mask.
- Dictionary requests send the selected term separately from compact nearby
  context and retry with progressively smaller payloads.
- If the Sarmad/gpt-oss-120b gateway is unavailable, the panel now starts with
  the technical reason (for example `HTTP 502 BadGateway`) and scrolls back to
  the top of the message.

## Download

Download `MagicMirror-v1.0.1-windows-x64.zip`, extract it, and run
`MagicMirror.Native.exe`.
