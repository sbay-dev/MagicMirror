# Magic Mirror v1.0.0

First public release of the Magic Mirror Windows overlay.

## Highlights

- Transparent draggable "glass" overlay for Windows.
- Capture text behind the overlay without capturing the overlay itself.
- UI Automation text extraction plus OCR fallback.
- Arabic-first translated document rendering with mixed English/acronym support.
- Reader-page mode with configurable background and ink colors.
- Click-to-select dictionary workflow with draggable dictionary panel.
- Sarmad / gpt-oss-120b gateway integration and explicit gateway failure
  reporting for dictionary requests.

## Download

Download `MagicMirror-v1.0.0-windows-x64.zip`, extract it, and run
`MagicMirror.Native.exe`.

## Notes

This release is Windows-first. Android/iOS/MacCatalyst scaffolding is retained
for later expansion, but runnable release assets are Windows-only.
