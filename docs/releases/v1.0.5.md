# Magic Mirror v1.0.5

This release keeps v1.0.4's dedicated Cloudflare Sarmad gateway and completes
the pivot from canvas-only translated blocks to native reader/editor surfaces
for trustworthy academic reading, selection, dictionary review, and provenance
auditing.

## Fixes

- The mirror status now shows the translation source:
  `Sarmad AI`, `MT fallback`, `mixed sources`, or `original fallback`.
- Dictionary responses are rendered as structured UI cards instead of raw
  Markdown text.
- Markdown tables returned by the model are converted into readable key/value
  rows and numbered alternative cards.
- Dictionary direction follows the first strong character per paragraph, so
  Arabic, English, and mixed technical lines follow W3C-style `dir=auto`
  behavior with Unicode bidi isolates.
- Dictionary cards use larger readable text and line height, calibrated
  click hit-testing, and a visible `جملة` action to review a full sentence in
  the dictionary.
- Word selection keeps the previous readable full-line rendering, while click
  handling returns to strict v1.0.1-style hit geometry: a word is selected only
  when the registered word region is actually hit.
- Repeated selected-word fallback/border boxes are disabled; selection now shows
  one direct marker for the active word/sentence.
- Original source/OCR text remains selectable too; source-side clicks use the
  capture-derived OCR block geometry from the earlier precise selection workflow.
- Translation-side selection is linked back to the original/OCR text: selecting
  a translated word marks the source OCR block and, when OCR line provenance is
  preserved, the corresponding source line. It does not claim source-word
  equivalence.
- Numbered dictionary alternatives include an `اعتمد` action. Choosing one
  updates the visible translation immediately and saves a local glossary-memory
  rule for future translation and dictionary prompts.
- Translation reveals returned text one Unicode text element at a time in the
  overlay, so the visible delivery is character-by-character instead of whole
  replacement batches.
- OCR translation captures default to a 2x high-quality screen image, giving
  OCR larger glyphs without changing the lightweight live-preview path.
- Tesseract tessdata is auto-discovered instead of pointing to a developer-only
  path; configure it manually only when auto-discovery cannot find `eng/ara`
  trained data.
- Free MT fallback is enabled for local reading continuity, labelled as
  non-academic quality when used, and should be disabled for confidential or
  publisher-style workflows.
- The mirror window can be enlarged/reduced progressively with mouse wheel or
  touch pinch; in translated mode, use `Ctrl` + wheel to resize while plain
  wheel keeps scrolling translated overflow.
- Wheel/pinch resize stays visually continuous, so the mirror no longer
  disappears and reappears during fast resizing.
- After releasing drag/resize/zoom, the settled mirror refreshes with a
  full-resolution capture instead of the lightweight preview frame.
- The visible translated reading path now uses native MAUI/WinUI text controls:
  a transparent mirror editor over the captured page plus a detachable reader
  window with independent text size, wrapping, copy, ink/background/opacity, and
  dictionary controls.
- Right-click menus in the mirror editor and detached reader include
  **معجم المحدد**, **نسخ**, and **تحديد الكل**.
- Selected reader/editor text maps back to its owning translated block and marks
  the corresponding OCR/source rectangle on the original capture.
- Dictionary output is split into **المعجم** for the linguistic answer and
  **التوثيق التقني** for source rectangles, translation source, local Merkle
  hashes/root/path, and proof notes.
- Dictionary lines follow the app target-language direction; technical proof
  rows remain LTR so hashes and paths stay readable.
- Translation starts Sarmad-only. If Sarmad cannot provide a usable aligned
  translation, the app asks whether to retry Sarmad, keep waiting/current text,
  or translate the whole capture with MT. MT is not mixed silently into Sarmad
  output.
- MT fallback is off by default, labelled non-academic when used, and documented
  as sending text to Google `gtx` / MyMemory.

## Scope

v1.0.5 is a personal/research academic reading-aid preview. It must not be
marketed as publisher-certified or as a replacement for human academic review.
Source-linking claims are limited to block/measured OCR-line provenance; the app
does not claim exact source-word equivalence after translation.

## Download

Use `MagicMirror-v1.0.5-windows-x64.zip` from the release assets, extract it,
and run `MagicMirror.Native.exe`.
